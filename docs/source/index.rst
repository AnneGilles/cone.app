===================================================
cone.app - Build Web Applications on top of pyramid
===================================================

``cone.app`` offers a web application stub for the pyramid framework.


Features:

* Base application layout
* Authentication integration
* Application model basics
* Authoring
* Workflows
* Common UI widgets
* AJAX helpers
* Pluggable
